# Raahi - Peer-to-Peer Ride Sharing Platform

A production-ready mobile application for intercity and hill-region transportation. Raahi connects private vehicle owners with passengers traveling toward similar destinations.

## 🚀 Tech Stack

### Mobile App
- **Framework**: React Native with Expo
- **Language**: TypeScript
- **State Management**: Zustand
- **Navigation**: React Navigation
- **Styling**: NativeWind (Tailwind CSS)
- **HTTP Client**: Axios
- **Maps**: Google Maps & react-native-maps
- **Real-time**: Socket.io
- **Authentication**: Firebase OTP
- **Notifications**: Firebase Cloud Messaging
- **Payments**: Razorpay

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Language**: TypeScript
- **Database**: PostgreSQL
- **Real-time**: Socket.io

## ✨ Features

### Passenger
- 🔍 Search & discover rides
- 📍 Real-time ride tracking
- 💳 Instant booking & payments
- 💬 Live chat with drivers
- ⭐ Ratings & reviews
- 🔔 Push notifications

### Driver
- 📤 Publish rides
- 📋 Manage booking requests
- 💰 Real-time earnings tracking
- 📊 Analytics dashboard
- ✅ Vehicle verification
- 🗺️ Route optimization

## 🛠️ Development Setup

### Prerequisites
- Node.js 16+
- npm or yarn
- Expo CLI
- PostgreSQL 13+

### Installation

```bash
git clone https://github.com/Vineet179/raahi-mobile.git
cd raahi-mobile
npm install
```

## 📱 App Screens

### Auth Flow
1. Splash Screen → 2. Onboarding → 3. Login → 4. OTP Verification → 5. Role Selection

### Passenger Flow
Home → Search → Ride Details → Booking → Payment → Success → My Trips → Chat → Ratings

### Driver Flow
Dashboard → Publish Ride → Manage Requests → Earnings → Analytics
